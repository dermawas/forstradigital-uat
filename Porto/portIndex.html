🗺️ Network Topology
WireGuard provides full-tunnel internet access for roaming devices. L2TP/IPSec connects Site A (Jakarta) and Site B (Cipanas) for internal resource access only — site clients use their own ISP for internet. Cross-VPN routing between WireGuard and L2TP clients is permitted via bidirectional routes on the OCI VM.
OCI VPN Hybrid Topology · WireGuard (full tunnel) + L2TP (site-to-site, internal only)

📜 Scripts (/usr/local/bin/)
ScriptPurpose
tg_send.shTelegram sender gateway. Takes message as $1, sends to group -1001781133257 with HTML parse mode.
tg_vpn_poller.shPolls Telegram bot for incoming commands (/peer add, /peer del, /vpn on, /vpn off, /list, /status). Runs every 30s via cron.
wg_peer_add.shCreates WireGuard peer: generates keypair, assigns next IP, writes client .conf (DNS=10.8.0.1, MTU=1280, AllowedIPs=0.0.0.0/0), generates QR, sends via Telegram. Updates wg_peers.map.
wg_remove_peer.shRemoves a WireGuard peer by name. Deletes from wg0, removes peers.d conf, updates wg_peers.map.
wg_watch.shMonitors WireGuard handshake freshness (threshold: 150s). Sends Telegram on connect/disconnect. Also alerts if a peer stays connected >4h (re-alerts every 1h). Protected by flock -n /run/wg_watch.lock.
wg_peer_loader.shLoads all peer stubs from /etc/wireguard/peers.d/ into wg0 on boot.
vpn_list.shLists all peers with name, IP, and current connection status. Called by /list Telegram command.
vpn_status.shShows detailed status of a specific peer. Called by /status Telegram command.
notify-app.shApache CustomLog pipe handler. Sends Telegram on KeePass (.kdbx) and GnuCash (.gnucash) WebDAV access. GET=opened, MOVE=saved & closed. Deduplicates GET within 60s.
pg_gnucash_monitor.shGnuCash PostgreSQL monitor daemon (systemd). Polls pg_stat_activity every 15s with in-memory state and 2-poll debounce.
backup-gnucash.shDaily PostgreSQL dump of gnucash database. Saves to /opt/gnucash-backups/ with timestamp. 7-day local retention.
sync-dropbox-weekly.shWeekly sync of latest GnuCash backup to dropbox:OCI-Backups/gnucash/ via rclone.
backup-keepass.shDaily KeePass vault backup at 03:30. Dated copy to /var/backups/keepass/ (14-day retention). Syncs to dropbox:OCI-Backups/keepass/.
backup-server-config.shDaily backup of /etc/wireguard and /usr/local/bin to dated .tar.gz archives in /var/backups/server-config/. Syncs to dropbox:OCI-Backups/wireguard/ and dropbox:OCI-Backups/scripts/. 14-day local retention.
vpn-health.shChecks all critical services every 5 minutes: wg-quick@wg0, apache2, postgresql, AdGuardHome, pg-gnucash-monitor. Also verifies WebDAV returns HTTP 401 and DNS resolves keepass.vpn correctly. Sends Telegram alert listing all failures.
wg-daily-summary.shSends daily WireGuard peer summary at 07:00.
wg-monthly-report.shSends monthly report on the 1st at 08:00.
check-wg-usage.shHourly check of WireGuard usage/bandwidth stats.
